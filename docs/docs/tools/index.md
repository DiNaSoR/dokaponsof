# Modding Tools
{: .no_toc }

A collection of tools for modding DOKAPON! Sword of Fury.
{: .fs-6 .fw-300 }

## Available Tools

### [Voice Pack Extractor](voice-extractor)
Extract and convert voice files from the game's `.pck` format to standard WAV files.
