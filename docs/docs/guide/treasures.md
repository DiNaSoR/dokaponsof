## Treasure System Overview

### Key Mechanics

- **Appraisal**: Treasures must be appraised at Item Shops (100G per item) to reveal their value
- **Effects**: In Remaster, effects activate immediately; in PS version, only after appraisal
- **Value**: Adds to total assets calculation
- **Stacking**: Some treasures can stack effects with multiple copies
- **Combination**: Some treasures combine into new items
- **Garmets**: Can be bought/sold at Garmets shop on Sundays
  - Appraised: Sells for half value
  - Unappraised: Sells for 100G

### Obtaining Methods

- **Village**: Excavation in villages
- **? Space**: Random events on ? spaces
- **Garmets**: Shop purchases (Sundays only)
- **Events**: Field events, rewards, casino, etc.
- **Monsters**: Drops from defeated monsters
- **Treasure Chests**: Magic Key spaces

## Treasure Combination Guide

### Skull Stone
Combine all 4 parts to create:
- **Something Fragment** (#22)
- **Something Part** (#23)
- **Something Piece** (#24)
- **Something Shard** (#25)

**Result**: Skull Stone (#21) - Value: 5,000,000G

### Shining Seasons
Combine all 4 seasonal parts to create:
- **Spring Romance** (#75)
- **Summer Memories** (#76)
- **Autumn Twilight** (#77)
- **Winter is Nabe** (#78)

**Result**: Shining Seasons (#74) - Value: 500,000G

### Dokapon Ball
Combine all 4 balls to create:
- **Weak Ball** (#95)
- **Meat Ball** (#96)
- **Strong Ball** (#97)
- **Food Ball** (#98)

**Result**: Dokapon Ball (#99) - Value: 99,999,999G
