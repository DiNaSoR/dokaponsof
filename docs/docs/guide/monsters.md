## Monster System Overview

### Key Mechanics

- **Elemental Weaknesses**: Monsters have elemental weaknesses that take **1.3x damage** from matching magic
- **Weapon Type Weaknesses**: Some monsters are weak to specific weapon types (PS version only)
- **Special Skills**: Each monster has unique abilities that affect combat
- **Drop Items**: Monsters can drop equipment, items, and treasures
- **Terrain Types**: Monsters appear in specific terrain types (Plains, Forest, Desert, Sea, etc.)

### Stat Abbreviations

| Abbreviation | Full Name | Description |
|--------------|-----------|-------------|
| Lv | Level | Monster level |
