# Modding Tools
{: .no_toc }

A collection of tools for modifying DOKAPON! Sword of Fury.

## Available Tools

### [Voice Pack Extractor](voice-extractor.html)
Extract and modify voice files from the game's `.pck` format.
- **Status**: Stable
- **Language**: Python
- **Input**: Voice-en.pck
- **Output**: WAV files
